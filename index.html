About this project

This is a live web demo of my Java To-Do List app. The original is a command-line program written in Java with full OOP architecture — classes for Task and TaskManager, file persistence with I/O, and a menu-driven interface.

This demo replicates the same logic in the browser so you can interact with it directly. Features include adding tasks, marking them complete, deleting them, filtering by status, and clearing finished ones — same as the real app.
